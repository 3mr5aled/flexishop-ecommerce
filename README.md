# FlexiShop — Minimalist Luxury E-Commerce Storefront

A high-fidelity, premium front-end e-commerce store built to showcase modern, framework-free client-side rendering, catalog filtering, persistent shopping cart dynamics, and a multi-step checkout form flow. Target-tailored for retail, WooCommerce, and Shopify-seeking freelance clients.

## 🚀 Live Demo Features

- **Minimalist Luxury Design**: High-contrast typography combining elegant Cormorant Garamond headings and clean Plus Jakarta Sans copy. Warm sand/beige palette highlighted with soft golden hues.
- **Dynamic Catalog Filters**: Live search input, category selection tags, and instant price slider matching that updates the catalog metrics in real-time.
- **Persistent Shopping Cart**: Fully interactive slide-out side drawer with local storage synchronization. Users can add items, increment/decrement quantities, delete items, and inspect subtotal/shipping adjustments that persist through browser reloads.
- **Custom Checkout Form Flow**: A modal overlay containing shipping addresses and credit card validation rules, culminating in a asynchronous processing state (spinner mockup) and custom invoice response screen.
- **Artistic Visual placeholders**: Pure CSS-drawn abstract vectors depicting designer lights, ceramics, and objects using modern gradients.

## 🛠️ Built With

- **HTML5**: Highly structured layout optimized for SEO and modern browser viewport testing.
- **CSS3 Variables & Grid/Flexbox**: Fully responsive styling configuring HSL tokens.
- **Vanilla JavaScript**: Lightweight client-side database management, filtering pipelines, and local storage state engines.
- **Lucide Icons**: External SVG-based asset loader.
- **Google Fonts**: Cormorant Garamond & Plus Jakarta Sans.

## 💻 Setup & Installation

1. Copy or clone this project folder into your environment.
2. Open `index.html` directly in your default web browser to play around.
3. Test persistent storage: add products to the cart, refresh the browser page, and open the cart drawer to see items preserved in the active state.
